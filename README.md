# console-number-guessing-game
A simple console game to guess randomly generated integer in the range.

## Technologies used
- NodeJs

## How to run the project
1. Install [Nodejs](https://nodejs.org/)(If not already installed).
2. Run command `npm install` to install the dependencies.
3. Check the demo video below to how to play the game.

[demo console number guessing game.webm](https://github.com/toukirahmeds/console-number-guessing-game/assets/10541098/8b126f45-0b9b-4bcd-be49-7ed823cd3e08)
